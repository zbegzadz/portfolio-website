Schiphol
Dashboard

Real-time vluchtinformatie op één overzichtelijk dashboard.

Over het project

Live data,
helder weergegeven.

Het Schiphol Dashboard toont actuele vluchtinformatie via een externe API. Gebruikers kunnen vluchten filteren op bestemming, status en tijdstip. Het dashboard is volledig responsive en werkt zowel op mobiel als op desktop.

Het project is gemaakt als leeropdracht gericht op het werken met externe API’s en het verwerken van asynchrone data in React met TypeScript.

Leeropdracht

Functionaliteiten

  1. 01

    Live vluchtdata

    Realtime koppeling met een externe API voor actuele vertrektijden, aankomsten en vluchtstatussen.

  2. 02

    Filteren & zoeken

    Filteren op bestemming, vluchtcode of status. Resultaten worden direct bijgewerkt zonder herladen.

  3. 03

    Responsive interface

    Volledig werkend op desktop en mobiel, met een aangepaste lay-out per schermgrootte.
